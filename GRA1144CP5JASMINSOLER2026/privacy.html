Privacy Statement

This privacy statement explains how information may be collected, used, and protected on this educational website project.

Overview

This website is a student project created for educational and training purposes only. It is not an official commercial website, and any content presented here is part of a classroom assignment designed to demonstrate web design, branding, layout, and front end development skills.

Information Collection

This project does not intentionally collect personal information from visitors. If this site were expanded to include features such as a mailing list sign up, contact form, or merchandise inquiry form, visitors might be asked to provide basic details such as a name or email address.

How Information Would Be Used

Any information entered into this educational website would only be used to demonstrate how a music artist website could handle fan communication, email subscriptions, event updates, or merchandise requests. Information would never be sold or shared for unrelated purposes.

Cookies and Tracking

This student project does not currently use cookies, tracking tools, or analytics software. In a real world website, cookies or analytics platforms might be used to improve site performance, understand visitor behavior, and create a better user experience.

Third Party Content

Some pages on this project may include third party content such as embedded videos, music, maps, or external links. Those platforms may have their own privacy practices, and visitors should review their policies separately, since this project does not control how outside services manage user data.

Security

Reasonable care should always be taken when entering information online. Because this is an educational project, visitors should avoid submitting sensitive personal, financial, or private information anywhere on the site.

Children’s Privacy

This educational website is not intended to collect information from children under the age of 13. No part of this classroom project is designed to knowingly request or store personal information from minors.

Changes to This Statement

This privacy statement may be updated if the project design, features, or class requirements change. Any updates would be reflected directly on this page.
